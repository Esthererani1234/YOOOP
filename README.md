# YOOOP

A fast, mobile-friendly marketplace storefront.

## Included

- Responsive homepage and navigation
- Product search
- Category filters
- Price and rating sorting
- Product cards and promotional sections
- Shopping cart with quantity controls
- Persistent cart using local storage
- Shipping and order-total calculations
- Static deployment support

## Preview with GitHub Pages

Open **Settings → Pages**, choose **Deploy from a branch**, select **main** and **/(root)**, then save.

## Current status

This is a working storefront prototype. Real payments, customer accounts, supplier imports, live inventory, order placement, and admin controls require secure backend services and API credentials.
